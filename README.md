# Networking Labs

A collection of hands-on networking labs built using Cisco Packet Tracer, Cisco IOS CLI, and real-world enterprise design principles.

Each lab includes:
- Topology diagrams  
- IP/subnet allocation  
- Switch/router configurations  
- Verification and troubleshooting  
- Packet Tracer `.pkt` files  
- Reflections and notes  

This repo serves as a growing technical portfolio showcasing practical networking skills.
